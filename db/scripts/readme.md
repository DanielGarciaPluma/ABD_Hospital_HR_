# 📜 Scripts SQL  
## Base de Datos – Módulo de Recursos Humanos

---

## 📖 Descripción

Esta carpeta contiene los scripts SQL utilizados para la configuración, creación y mantenimiento de la base de datos del módulo de Recursos Humanos.

Los scripts permiten implementar la estructura del sistema de manera controlada y replicable en distintos entornos (desarrollo, pruebas o producción).

---

## 🎯 Objetivo

- Facilitar la creación de tablas y estructuras necesarias.
- Permitir la inicialización del entorno de base de datos.
- Ejecutar configuraciones específicas del sistema.
- Apoyar tareas de mantenimiento y actualización.

---

## ⚙️ Alcance

Los scripts pueden incluir:

- Creación de esquemas y tablas.
- Definición de claves primarias y foráneas.
- Creación de índices.
- Inserción de datos iniciales.
- Modificaciones estructurales.

---

## 🏗️ Importancia

El uso de scripts versionados garantiza:

- Reproducibilidad del entorno.
- Control de cambios estructurales.
- Orden en el desarrollo del proyecto.
- Mayor seguridad en procesos de implementación.

---

## 📌 Consideraciones

Todos los scripts deben:

- Seguir la nomenclatura establecida en el proyecto.
- Estar documentados internamente.
- Ser probados antes de ejecutarse en entornos productivos.
- Mantener coherencia con el modelo de datos definido.
