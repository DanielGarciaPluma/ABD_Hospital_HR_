# 📊 Diagrama del Modelo de Datos  
## Base de Datos – Módulo de Recursos Humanos

---

## 📖 Descripción

Esta carpeta contiene el diagrama entidad–relación correspondiente al módulo de Recursos Humanos del sistema hospitalario.

El archivo `Diagrama.drawio` representa gráficamente la estructura del modelo relacional, incluyendo las entidades principales y sus relaciones.

---

## 📂 Archivo Incluido

| Archivo | Descripción |
|----------|------------|
| `Diagrama.drawio` | Modelo visual del esquema de base de datos desarrollado en Draw.io. |

---

## 🎯 Objetivo del Diagrama

El diagrama tiene como finalidad:

- Representar visualmente las entidades del sistema.
- Mostrar las relaciones entre tablas.
- Identificar claves primarias y foráneas.
- Facilitar la comprensión de la arquitectura del modelo de datos.

---

## 🏗️ Importancia del Modelo Visual

La representación gráfica del esquema permite:

- Validar la coherencia estructural del diseño.
- Detectar posibles redundancias o inconsistencias.
- Facilitar el mantenimiento y futuras ampliaciones.
- Apoyar la documentación técnica del proyecto.

---

## 📌 Consideraciones

El modelo fue diseñado siguiendo principios de normalización y buenas prácticas en bases de datos relacionales, garantizando integridad, escalabilidad y consistencia en la gestión del personal hospitalario.
