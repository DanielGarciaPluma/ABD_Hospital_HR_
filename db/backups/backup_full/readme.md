## 🌟 Descripción

Esta carpeta contiene el **respaldo completo (Full Backup)** de la base de datos correspondiente al módulo de Recursos Humanos.

El backup completo incluye tanto la estructura como los datos almacenados, permitiendo la restauración total del sistema en caso de fallos, errores críticos o pérdida de información.

---

## 📦 ¿Qué incluye este respaldo?

- 🏗️ Estructura de tablas
- 🔑 Claves primarias y foráneas
- 📊 Datos almacenados
- 📌 Índices y restricciones
- 🔐 Configuración básica relacionada con la integridad

---

## 🎯 Objetivo

Garantizar la disponibilidad y recuperación de la información crítica del personal hospitalario, asegurando:

✔ Continuidad operativa  
✔ Protección ante fallos técnicos  
✔ Seguridad de la información  
✔ Integridad del sistema  

---

## 🔄 Importancia del Backup

En sistemas administrativos hospitalarios, la información del personal es un activo fundamental.  
Contar con respaldos completos permite minimizar riesgos y asegurar la estabilidad del sistema ante cualquier eventualidad.
